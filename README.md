# 相続登記の見本

相続登記の申請に必要な書類一式の見本です。Astro で構築しており、ブラウザから印刷してそのまま使える体裁になっています。

**記載されている人名・住所・不動産情報はすべて架空のものです。**

## 収録書類

- 遺産分割協議書
- 登記申請書（所有権移転）
- 相続関係説明図
- 委任状（所有権移転）

## 使い方

```bash
npm install
npm run dev
```

各ページをブラウザで開き、印刷（Ctrl+P / Cmd+P）すると A4 用紙に出力できます。

`src/data/` 内の JSON ファイルを書き換えれば、自分のケースに合わせた書類を生成できます。
